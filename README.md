# וועדת ההשקעות — סימולציית שוק ההון

סימולציית למידה אינטראקטיבית בעברית (RTL) על שוק ההון, למשקיעים מתחילים. הלומד נכנס לתפקיד חבר ועדת
השקעות, מקבל תקציב של מיליון ש״ח, ומקבל החלטות בסיסיות בליווי **עלמה** — יועצת דיגיטלית — תוך שהוא לומד
כיצד שוק ההון מחבר בין משקיעים, חברות וממשלות, ומה הקשר בין סיכון לתשואה.

> המרה של גרסת ה‑HTML המקורית לאתר מבוסס **React + TypeScript + Vite**.

## טכנולוגיות

- **React 18** + **TypeScript**
- **Vite** (כלי build ושרת פיתוח מבוסס Node.js)
- CSS גלובלי עם design tokens (מערכת העיצוב "Capital Insight System")
- גופן **Heebo** (Google Fonts)

## הרצה מקומית

דרושה התקנת [Node.js](https://nodejs.org) (גרסה 18 ומעלה).

```bash
npm install      # התקנת תלויות
npm run dev      # שרת פיתוח (http://localhost:5173)
npm run build    # בנייה לפרודקשן (tsc + vite build → dist/)
npm run preview  # תצוגה מקדימה של ה-build
npm run typecheck# בדיקת טיפוסים בלבד
```

## מבנה הפרויקט

```
public/media/          נכסי וידאו ותמונות (סרטונים, עלמה, דמויות, מטבע)
src/
  main.tsx             נקודת הכניסה
  App.tsx              מעטפת — ניתוב בין 8 השלבים + שכבת וידאו מלא-מסך
  styles.css           כל ה-CSS (design tokens, רכיבים, רספונסיב)
  types.ts             טיפוסים משותפים
  data/content.ts      תוכן: שאלות, משתתפים, דירוג, תיק, מילון מושגים
  state/SimContext.tsx  ניהול מצב (שלב, מדים, החלטות, הודעות עלמה, toast)
  lib/                 utils + אייקוני SVG
  components/
    Opening, Header, Meter, Stepper, AlmaRail, VideoOverlay, NavBar
    steps/             8 רכיבי השלבים (Intro, Market, Parts, Trade,
                       Connect, Risk, Folio, End)
```

## מבנה הלמידה (8 שלבים)

1. **כניסה לתפקיד** — סרטון היכרות עם עלמה + מטבע כניסה
2. **מהו שוק ההון** — סרטון הסבר + בדיקת ידע
3. **המשתתפים** — סרטון + משימת התאמה
4. **מניה מול אג״ח** — החלטת השקעה + משוב מובנה ועדכון מדים
5. **איך הכול מתחבר** — דיאגרמת זרימה + בדיקת הבנה
6. **סיכון מול תשואה** — משימת דירוג
7. **בניית תיק** — הקצאת התקציב + ניתוח מסכם
8. **רפלקציה וסיכום** — השלמת משפט + מילון מושגים
